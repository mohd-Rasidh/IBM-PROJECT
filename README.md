📚 IBM Project – Accessible Learning Platform
📌 Project Overview

This project is developed as part of the IBM SkillsBuild / Naan Mudhalvan initiative.
The main goal of this project is to create an AI-powered accessible learning platform that helps visually and hearing-impaired students access educational content easily.

Traditional e-learning systems often lack accessibility features. This project focuses on converting educational materials such as videos, images, and documents into accessible formats including captions, audio descriptions, and assistive learning aids.

🎯 Objectives

Provide inclusive learning support for visually and hearing-impaired students

Convert educational content into accessible formats

Improve the learning experience through AI technologies

Build a user-friendly and scalable platform

🚀 Features

🎥 Automatic Video Captioning

🔊 Audio Description for Visual Content

🧏 Assistive Support for Hearing-Impaired Students

📄 Accessible Document Conversion

🧠 AI-based Content Processing
